SGML

SGML (Standard Generalized Markup Language) is a standard for how to specify a document markup language or tag set. Such a specification is itself a document type definition (DTD). SGML is not in itself a document language, but a description of how to specify one. It is metadata.


SGML is based on the principle that documents have structural and other elements that can be explained without reference to how such elements should be displayed. The actual display of such file may change, depending on the output media and style preferences.

The language that this Web sites use, Hypertext Markup Language (HTML), is an example of an SGML-based. There is a definition for HTML. In today's networking ecosystem, many documents are being described with the Extensible Markup Language (XML) which is a data description language that uses SGML principles.

Some advantages of documents based on SGML are:

  1. These are made in terms of document structure rather than appearance characteristics.
  2. They will be more portable because an SGML compiler can understnad any document by its document type definition .
  3. Documents originally used for the print media can be re-used for other media, such as the computer display screen.

Some disadvantages of documents based on SGML are:

  1. Linking is complex
  2. writing SGML is very hard

SGML is based on earlier general markup languages developed by IBM.

  • In this family of markup languages, Standard Generalized Markup Language (SGML) is the parent.
  • HyperText Markup Language (HTML) is a child, or application, of SGML.
  • Extensible Markup Language (XML) is a cousin to HTML and a nephew to SGML.

XML

Extensible Markup Language is used to describe data. The XML standard is a flexible way to create information formats and electronically share structured data via the public Internet, as well as via corporate networks.


XML data is known as self-defining, meaning that the structure of the data is stored with the data, so when data comes there is no need for a pre-made structure for it, it is understood within the XML. The XML format can be used by any individual who wants to share content in a specific way. XML is actually a simpler and easier-to-use subset of the Standard Generalized Markup Language (SGML).


Applications for XML are endless. For example, computer makers might agree upon a standard or common way to describe the information about a computer product (processor speed, memory size, and so forth) and then describe the product information format with XML code. Such a standard way of describing data would enable a user to send an intelligent agent (a program) to each computer maker's Web site, gather data, and then make a valid comparison.

SUMMARY

SGML
Standard Generalized Markup Language, an international standard for defining methods of encoding electronic texts to describe layout, structure, syntax, etc., which can then be used for analysis or to display the text in any desired format.
XML
XML is a file extension for an Extensible Markup Language (XML) file format used to create common information formats and share both the format and the data on the World Wide Web, intranets, and elsewhere using standard ASCII text. XML is similar to HTML
